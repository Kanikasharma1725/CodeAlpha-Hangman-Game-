# Interactive Hangman Game in Python

A clean, modular, and dynamic console-based Hangman game built using Python. The project features multiple categories, a built-in hint system, dynamic ASCII art stages, and real-time score tracking.

## Features

- **Multiple Categories:** Players can choose from 5 different categories: Fruits, Animals, Countries, Colors, and Sports.
- **Dynamic Word Pool:** Each category contains 10 distinct words that are picked completely at random using Python's `random` module.
- **Built-in Hints:** Every word comes with a unique contextual hint displayed at the start of the round to assist the player.
- **Visual Hangman Stages:** Features dynamic ASCII art that progressively updates with every incorrect guess.
